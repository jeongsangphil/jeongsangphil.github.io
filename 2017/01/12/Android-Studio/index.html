Activity

Activity : 화면 하나 전체

ActivityFragment : 화면에 표시할(갖다붙이는) 것들의 그룹, 화면 바꾸는 것.

StartActivity(Intent) : 화면 전환 (main Activity -> detail Activity)

Intent

Intent : Activity간 화면전환할 때 통신을 위한 방법. ex) db 로그인할때 값 전달, session

Explicit Intent(유기명) : 실제 이름 만들고 보내는사람 받는사람 존재

Implicit Intent(무기명) : 해야할 일은 정해져있으나 어떤 대상에 제한되지 않음

ex) 어플에서 전화번호 누르면 전화앱 여러개 선택할수있음, sms, 메일주소 등

Broadcast Intents

action view/ send 는 지도누르면 보여주고, 보내고 -> broadcast Intents

방송(메세지) 가 일어남 ,ex) 컴퓨터키고 시간을 체크하던, 업데이트 체크하던.

안드로이드 내에서는 항상 공유상태가 있음, 통신

ex) 액션뷰로 geo를 보내면 지도앱들이 반응을하는 broadcast(메세지 전송) 이 일어남

intent filter에서 broadcast receiver로 어떤액션을 원하는지 선택함 티비볼때 원하는 채널 설정, 한개이상

필터를이용해서 어떤상황이되면 메세지를보내주는 , 전체채널을 쏴주고, 원하는거 골라서.. ex) 케이블 tv 원하는채널
