      CSS코드(태그)
      css란? Cascading Style Sheets (CSS)는 documents가 사용자에게 어떻게 보여질가를 기술하는 언어이다.

      css를 사용하려면 html에서 document를 만들어야한다. 위에 <!DOCTYPE html>적기
      Document는 마크업 언어로 사용된 정보의 집합이다.

      html에 css파일을 연결하려면 head위에 link를 선언해야한다.
      ex)







      /**/ : css에서의 주석

      css파일에서 html을 꾸밀 것을 선언하면 된다.
      ex) span{
      font-family: Arial;
      color: blue;
      } / html에서 css 상단 양식을 불러와서 쓰려면, 을 선언하면 된다. /

      {}괄호 내에 들어갈 꾸미는 코드
      border: 1px dashed(solid) blue;
      text-decoration: none;
      font-family: Verdana, sans-serif; /Verdana폰트가 있으면 사용, 없으면 백업 폰트로 sans-serif사용/
      ex) div{ / div selector /
      border-color: #6495ED; /16진수로 색 표현, 그냥 영어로 써도됨 ex) blue, black ../
      border-radius: 100%; /border는 테두리, 100%면 원형으로 표시/
      }

      universal selector () : html 전체를 꾸밀때 전체라는 표시
      ex)       {
      font-size: 10px;
      }
      text-align: center/left/right /텍스트 정렬/

      parent, children, siblings : html태그들을 tree라고 생각
      ex)

      /여기서는 div가 parent, p이 child?, 2개의 p를 siblings?/




      div p { /css에서 div안의 p만 꾸미고 싶을때/
      color: blue;
      }
      direct-children (>) : 직접적인 selector
      ex) div > p { }

      class : 클래스로 나누어서 클래스별로 꾸미는 양식 다르게.
      ex) .red{ color: red }

      /h1말고도 여러곳에 사용가능 html, div등/
      id : 각 아이디별로 꾸미는 양식 다르게
      ex) #rogue{ font-size: 10px }

      /p말고도 여러곳에 사용가능 div, p 등/

      pseudo-class_selector: 방문한 링크/방문하지않은 링크 색으로 구분?
      link는 를 말한다.
      nth-child(번호n){ } : 원하는 번째의 child만 꾸미기
      ex) p:nth-child(2){ } /2번째 paragraph만 따로 꾸미기/

      css positioning : css box model에서 바깥부터 margin(transparent), border(경계), padding(국경), content(innermost layer)
      줄여서 방향별로 tm/tb/tp, rm/rb/rp … 나타낼수있음
      margin: 외부 여백
      ex) margin: 1 2 3 4; /1~4는 상단부터 시계방향대로 설정, margin-top/right/bottom/left 로 사용해도됨
      margin: 0 auto는 자동으로 가운데로 정렬해줌
      border : 경계(테두리)
      ex) border: 1px solid blue; /      순서대로 두께(width), 스타일(style), 색상(color)이다./
      /       스타일은 “solid”, “dashed”, “dotted”로 지정 가능 /
      padding : 내부 여백으로 자간이라고 생각하면됨
      ex) padding: 1 2 3 4; /      margin처럼 방향대로, margin-top/right/bottom/left로 사용/
      /       margin값, border값, padding값 전부 고려해서 html에 적용*/

      float: right/center/left /지정한 방향부터 순서대로 위치하게됨 ex) 블록/
      display: 어떻게 표시할지 지정
      ex) inline/ block/ inline-block/ none;
      /block은 세로로 쌓는 것, inline은 수평으로 놓는것 , inline-block은 block으로쌓다가 중간에 inline형식을 쓸때, none은 기본(왼쪽부터)/

      clear: right/left/both; /원하는 방향에 float지정해둔것을 초기화?한다는 의미랑 비슷, clear하고나서 다음줄부터는 지정안된 기본상태/
      position: absolute/static(default)/relative/fixed /위치 지정, fixed의경우 스크롤 업/다운해도 위치고정/
