블로그 사용 방법

git bash들어가서
blog 폴더로 이동
cd blog
블로그 글 새로 작성
hexo new post '블로그 글 제목'
블로그 글 이름 변경
mv 변경전파일이름.md 변경후파일이름.md

현재 폴더 내 모든항목 보기
ls
블로그 글 폴더 위치
cd blog/source/_posts/

블로그 글 삭제는 blog/source/_posts/ 폴더에 들어가서
rm 삭제할파일이름.md
상위폴더로 이동
cd ..

블로그 글 업로드 과정
hexo clean
hexo d -g

서버 start
hexo server
